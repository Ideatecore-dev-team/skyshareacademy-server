# msim-server-new

Backend API server for SkyShare Academy — built with **Node.js v22**, **TypeScript**, **Express**, and **Prisma**. Implements secure, scalable REST API architecture following **SOLID principles** and **Clean Code** best practices.

## Features

- ✅ Written in TypeScript
- ✅ Environment-specific configuration with `.env` files
- ✅ Prisma ORM with PostgreSQL
- ✅ Middleware: Helmet, CORS, HPP, Cookie Parser
- ✅ JWT authentication
- ✅ Rate limiting (DDOS protection)
- ✅ Centralized error handling with `Zod` validation
- ✅ Logging with Winston + daily rotate
- ✅ Biome for linting & formatting
- ✅ Vitest + Supertest for E2E/API testing
- ✅ OpenAPI documentation with Swagger UI
